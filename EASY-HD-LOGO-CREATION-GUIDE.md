### Step 1: Setup Canvas
1. Go to **https://www.photopea.com**
2. Click "Create New" ‚Üí "Custom Size"
3. Set dimensions: **2048 x 2048 pixels**
4. Background: **Transparent**

1. File ‚Üí Open ‚Üí Upload your current Logo.png

4. Go back to your 2048x2048 canvas

6. Resize to fit (don't worry about quality yet)

### Step 3: Create HD Neon Text
1. Select **Text Tool** (T key)
2. Click on canvas and type "LITTLE LATTE LANE"
3. **Font Settings**:
   - Font: Bold, modern font (Arial Black, Impact, or similar)
   - Size: **200-250px**
   - Color: **#00FFFF** (bright cyan)
   - Alignment: Center

4. For "Caf√© and Deli" subtitle:
   - Create new text layer
   - Size: **80-100px**
   - Color: **#FF00FF** (bright magenta)
   - Position below main text

### Step 4: Add Neon Glow Effects
1. Right-click text layer ‚Üí "Blending Options"
2. Add these effects:

**Outer Glow:**
- Blend Mode: Screen
- Opacity: 80%
- Color: #00FFFF (cyan)
- Size: 15-20px
- Spread: 10%

**Inner Glow:**
- Blend Mode: Screen
- Opacity: 60%
- Color: #FFFFFF (white)
- Size: 8-12px

**Drop Shadow:**
- Color: #00FFFF (cyan)
- Opacity: 40%
- Distance: 0px
- Size: 25px

### Step 5: Export HD Logo
1. File ‚Üí Export As ‚Üí PNG
2. Quality: **100%**
3. **Keep transparency checked**
4. Save as "Logo-HD.png"
